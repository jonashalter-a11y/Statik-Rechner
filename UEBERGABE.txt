PROJEKT-ÜBERSICHT
================================================================================

Name: SIA 265 Holzbau Rechner mit SIA 261 Einwirkungen
Status: Produktiv — Dual-Norm-System vollständig implementiert
Backend: Node.js/Express, SQLite, PORT 3002
Frontend: React + TypeScript, Vite, Zustand (State Management)
Letzter Stand: Anhang C SIA 261 (Tab. 31–49) mit SVG Gebäudeskizzen

PROJEKTZIELE ERREICHT
================================================================================

✅ SIA 265:2021 Holzbau-Nachweise
   - Biegung, Schub, Knicken, Kippen, Torsion, Tragfähigkeit
   - 138 Kapitel hierarchisch, 13 Nachweise mit Verifikationsformeln
   - Automatische Materialkennwert-Einfüllung nach Holzklasse
   - η-Berechnung (Ausnutzungsgrad) mit Pass/Fail-Status

✅ SIA 261:2020 Einwirkungen auf Tragwerke
   - 90 Kapitel + 6 Verifikationen (Schnee, Wind, Nutzlast, Erddruck)
   - Schneelast-Nachweis: μᵢ·Cₑ·Cₜ·sₖ mit Bezugshöhe h0
   - Wind-Staudruck + Druckbeiwerte
   - 19 Anhang-C-Tabellen (Tab. 31–49) mit Gebäudeskizzen (SVG)
   - Ergebnisse als kN/m² oder kN (nicht η)

✅ Dual-Norm-System
   - Norm-Switcher in Header (SIA 265 | SIA 261 | EC5)
   - Norm-spezifische Chapters + Verifications auf Backend + Frontend
   - Automatische UI-Anpassung (Holzart/Klasse ausgeblendet bei SIA 261)
   - Cache-basiertes Laden ohne Neuberechnung

✅ Datenbankstruktur
   - norm_id Foreign Key auf alle Tabellen (chapters, verifications, db_tables)
   - Wood Types + Classes mit Materialkennwerten
   - Parametrisierbare Verifikationsformeln (compute_expr JavaScript)

✅ Frontend-Features
   - Resizable Columns (Sidebar, Nachweis-Panel, Ausdruckprotokoll)
   - PDF-Export mit jsPDF + html2canvas
   - Interaktive Gebäudeskizzen (SVG) in Datenbank-Tabellen
   - Live-Berechnung bei Variablen-Änderung
   - Ausdruckprotokoll mit Formeln + Wertetabellen

BACKEND (server/)

Dateistruktur:

  ├── index.js               [Express API, CORS, JSON Parsing]
  ├── db.js                  [SQLite Init, Schema, Seed-Runner]
  ├── seed-chapters.js       [SIA 265 — 138 Kapitel, 27 mit Verifikationen]

API-Endpoints:
  GET    /api/chapters?norm=sia261          [Kapitelstruktur filtern]
  GET    /api/verifications?norm=sia261     [Nachweise mit Formeln+Variablen]
  GET    /api/db-tables?norm=sia261         [Tabellen (Wind, Schnee, etc.)]
  GET    /api/db-tables/:id                 [Einzelne Tabelle mit Skizze]
  POST   /api/chapters, PATCH, DELETE       [CRUD für Kapitel]
  POST   /api/verifications, PATCH, DELETE  [CRUD für Nachweise]
  GET    /api/wood-types, /wood-classes     [Material-Optionen]

SQLite Schema:
  chapters
    ├── id, norm_id, number, title, description, parent_id
    └── Indices: (norm_id, number), (parent_id)

  verifications
    ├── id, norm_id, chapter_id, title, compute_expr, formula_latex
    └── Indices: (norm_id, chapter_id)

  variables
    ├── id, verification_id, name, label, unit, default_value, type, options
    └── type: 'number' | 'dropdown'

  db_tables
    ├── id, norm_id, category, title, description, headers (JSON), rows (JSON)
    └── description: kann "shape:xyz|text" enthalten für SVG-Skizzen

  wood_types
    ├── id, name, label, sort_order

  wood_classes
    ├── id, wood_type_id, name, label

  wood_properties
    ├── id, wood_class_id, key (f_m_k, f_v_k, E_0_mean), value, unit

3. VERIFIKATIONS-BERECHNUNG
─────────────────────────────────────────────────────────────────────────────
   Compute-Expr (JavaScript String):
     SIA 265: (function(){ var eta = f_m_d / sigma_m_d; return eta; })()
     SIA 261: (function(){ var sk = Math.max(0.9, (1 + h0²/350) * 0.4); return mu_i * Ce * CT * sk; })()

   evalFormula(expr: string, vars: Record<string, number>)
   ├─ substituteFormula(expr, vars)      [f_m_k → 24, etc.]
   └─ Function(expr)() eval              [Return numerisches Ergebnis]

   computeVerification(v: Verification)
   └─ eta ≤ 1.0 ? passed = true : false  [SIA 265 only]

================================================================================
BERECHNUNGSFORMELN (Beispiele)
================================================================================

SIA 265 — Biegung um x-Achse:
  η = σ_m_d / f_m_d
  Variablen: sigma_m_d (MPa), f_m_d (MPa)
  Nachweis erfüllt wenn η ≤ 1.0

SIA 261 — Schneelast auf Flachdach:
  s_k = μᵢ · Cₑ · Cₜ · sₖ(h₀)
  sₖ(h₀) = max(0.9, (1 + h₀²/350) * 0.4) kN/m²  [Anhang D SIA 261]
  Variablen: mu_i (Form), Ce (Exposure), CT (Thermal), h0 (Bezugshöhe m)
  Ergebnis in kN/m² (keine η-Berechnung)

SIA 261 — Wind Staudruck:
  qₚ = 0.5 · ρ · v_p² = cf · qₑ
  qₑ = 0.5 · ρ · v_ₑ² (Referenzdruck)
  Variablen: rho (Luftdichte), v_p (Spitzenwert m/s), cf (Druckbeiwert)
  Ergebnis in kN/m²
